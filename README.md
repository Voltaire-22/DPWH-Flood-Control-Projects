# DPWH Flood Control Projects (2018-2025)
## Introduction
This data piqued my interest to do a data analysis projects, as with the major issues that is currently happening in the Philippines about the "Ghost Projects", the goal of this project is to analayze the current performance of DPWH on delivering infrastructure (road) projects across the country.

Data Source: https://www.kaggle.com/datasets/bwandowando/dpwh-flood-control-projects
